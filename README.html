HOW TO BUILD OR PREVIEW IT LOCALLY?

Please read the following instructions to build or preview the project locally.

Prerequisites

  • Note: If you don’t have git installed, you can download it from here.

  • Note: If you don’t have node.js installed, you can download it from here.

  • Note: If you don’t have npm installed, you can download it from here.

How to Build or Preview the Project Locally

Installing the project(First time only)

  1. Open your terminal.
  2. Install the packages: npm install
  3. Run the project: npm start
  4. Open your browser and go to http://localhost:4173/

Building the project

  1. Open your terminal.
  2. Build the project: npm run build
  3. The build will be available in the dist folder.
  4. You can update to the server and deploy the project for production.

Previewing the project

  1. Open your terminal.
  2. Build the project: npm run build
  3. Run the previewing code: npm run preview
